# TeLLevoAPP

## Descripción
Este proyecto móvil es el encargo semestral encargado para la asignatura de Programación de aplicaciones móviles. Te LLevoAPP tiene el fin de conectar a los alumnos que disponen de vehículo con los que no poseen para coordinar un retorno a sus hogares en un tiempo más corto entre quienes comparten cercanía en sus hogares con quienes tienen movilización propia.
